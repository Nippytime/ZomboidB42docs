# AI Usage Notes

This repository is intended to be used as the source of truth for Project Zomboid Build 42 modding assistance.

AI assistants must inspect `_ai_context/PROJECT_RULES.md` and `_ai_context/REPO_INDEX.md` before answering technical questions about this repository.

Important rules:

- Search repo references before answering.
- Prefer decompiled Java and vanilla Lua examples over memory.
- Verify Java-to-Lua exposure before recommending Java methods from Lua.
- Verify client/server/shared context.
- Verify Build 42 behavior before applying Build 41 assumptions.
- Do not invent APIs, events, methods, fields, globals, return types, or registry syntax.
- Mark confidence as Confirmed, Likely, or Uncertain.
- Cite source files when possible.
